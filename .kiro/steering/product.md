# Product Overview

Offline-first AI coding agent that runs entirely on the user's machine. Code and data never leave the local environment.

## Core Capabilities

- **Local AI Execution**: Run AI coding agents without cloud dependencies
- **Filesystem Integration**: Read, write, edit, and search files with agent actions
- **Permission System**: User approval required for agent actions (sandboxed control)
- **Multi-Workspace**: Support for multiple isolated workspace contexts
- **Provider Flexibility**: Support multiple LLM providers with credential management

## Target Use Cases

- Privacy-sensitive development workflows
- Offline coding assistance
- Local-first AI pair programming
- Secure enterprise environments requiring data isolation

## Value Proposition

- **Privacy by Design**: All code, context, and AI interactions stay local
- **User Control**: Explicit permission gates for all file system operations
- **Offline Capability**: Full functionality without internet connectivity
- **Extensible**: Modular provider system supports various LLM backends
